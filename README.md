# Website Project

This project is part of the Web Development Basics course at Laurea University of Applied Sciences.

The goal of the project was to create and publish a multi-page website using HTML and CSS. The website follows the requirements given in the final assignment.

## Description

This website was built using native HTML and CSS based on the skills learned during the course workshops.

The website includes:

- At least four separate pages
- Clear and consistent navigation on all pages
- Responsive layout that works on mobile and desktop
- Semantic HTML structure
- CSS styling using Flexbox, Grid, and media queries
- Images, lists, tables, and a contact form
- Basic interactivity for mobile navigation

The homepage is named index.html as required.

## Pages

- Home (index.html)

- Projects
- Contact

Each page is accessible through the navigation menu.
